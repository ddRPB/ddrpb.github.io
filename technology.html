Technology

  • Continuous development life-cycle
  • Developed and tested for Linux servers (preferably Debian)
  • RPB-portal can be used with standard browsers such as Firefox, Chrome, Internet Explorer
  • RPB-client can be used with 64-bit desktop computer (Windows 10, Linux)
  • PostgreSQL as database back-end
  • optional [DICOM viewer]: Java Runtime Environment (JRE)

Our development toolbox

  • Java (Tomcat, JSF, Spring, Hibernate ORM, Primefaces)
  • Python (PyQt, SqlAlchemy, pydicom, pysimplesoap, requests)
  • Lua (Conquest DICOM scripting)
