Online application suite for translational research

Features and Modules

  • Manages multi-centre clinical trials
  • Pseudonymise participating patients
  • Randomises patients in randomised trials (permutated block + strata)
  • Manages collection of medical imaging and treatment plans in DICOM/ DICOM-RT format
  • Handles DICOM data de-identification and RTSTRUCT ROI naming harmonisation
  • Embedded DICOM viewer with DICOM-RT plugin
  • Enables data collection in clinical trilas using electronic case report forms (eCRFs)
  • Patient reported outcomes (ePRO) and clinician outcome assessment (eCOA) data collection
  • Allows semantic annotation of eCRFs
  • Supports storage for laboratory assays data
